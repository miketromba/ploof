This is the Ploof monorepo, managed with [Turborepo](https://turbo.build) and Bun workspaces.

## What's inside

| Package | Path | Description |
| :-- | :-- | :-- |
| **`@miketromba/ploof`** | [`packages/cli`](packages/cli) | The published CLI — generate & edit images, video, and audio from OpenAI and fal.ai. See its [README](packages/cli/README.md). |
| **`web`** | [`apps/web`](apps/web) | The marketing + docs site for [ploof.sh](https://ploof.sh), built with Astro. Every asset on it was generated by the CLI. |

## Develop

```bash
bun install            # install all workspaces

bun run dev            # run the website (apps/web) at localhost:4321
bun run build          # build everything via Turborepo
bun run lint           # lint (CLI)
bun run typecheck      # typecheck (CLI)
bun run test           # test (CLI)

# work on just the CLI
bun --cwd packages/cli run ploof -- --help
bun --cwd packages/cli run test
```

Turborepo orchestrates per-package tasks; see [`turbo.json`](turbo.json).

## Releasing the CLI

The CLI publishes to npm from GitHub Actions on a `v*` tag via npm Trusted
Publishing (see [`.github/workflows/publish.yml`](.github/workflows/publish.yml)).
Full details live in [`packages/cli/SPEC.md`](packages/cli/SPEC.md).
